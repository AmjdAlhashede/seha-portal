# منصة صحة

منصة إلكترونية تخدم القطاع الصحي في المملكة العربية السعودية

## البنية

```
Seha/
├── index.html          # الصفحة الرئيسية
├── assets/
│   ├── css/           # ملفات التنسيق
│   ├── js/            # ملفات JavaScript
│   ├── images/        # الصور والأيقونات
│   └── fonts/         # الخطوط
└── README.md
```

## التشغيل

افتح ملف `index.html` في المتصفح مباشرة، أو استخدم خادم محلي:

```bash
python -m http.server 3000
```
